# 開発スケジュール概要

## キックオフと体制
- 2025-10-27（月）にキックオフ。
- 2週間スプリント、編成はフロントエンド1名・バックエンド1名・プロダクト/UX1名。
- 現行インフラと Node.js/Tailwind スタックを継続利用。
- RESTチャットは既存API改修で実現し、評価シミュレーターは新規ドメインモジュールで対応。
- UIファーストでモックデータを用いた画面先行実装を行い、後段でAPIと計算ロジックを統合。

## リスクと対策ハイライト
- 税制・金融前提の調整が必要な場合はフェーズ3で税務・法務レビューを追加。
- `.env` 再編によるCI/CD失敗リスクに備え、フェーズ4冒頭でCI更新を実施。
- 年末年始の工数減少を想定し、フェーズ5に1週間のバッファを確保し機能フリーズ日を厳守。
- チャットポーリング負荷にはフェーズ3で計画策定、フェーズ5で検証を実施。
